# Android Academy

Android Academy is a modern learning app designed to help you master Android development step by step using interactive quizzes.

---

## Features

- Structured learning by categories and subcategories  
- Unlock system – progress through topics step by step  
- Progress tracking with percentage indicator  
- Quiz-based learning approach  
- Clean UI built with Jetpack Compose  

---

## What you’ll learn

- Kotlin (Loops, Collections, Advanced)
- Coroutines & Flow  
- Jetpack Compose  
- Room Database  
- Navigation  
- Dependency Injection  
- State Management  
- Testing  

---

## Tech Stack

- Kotlin  
- Jetpack Compose  
- MVVM Architecture  
- Navigation Component  
- SharedPreferences (progress persistence)
